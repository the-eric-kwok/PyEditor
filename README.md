# PyEditor
Simple text editor made with python

## Run
Run in terminal
```
python3 editor.pyw
```
## Build
- ### On macOS
    
    **Dependency**
    
    1. py2app
    
       Install by `pip3 install py2app`
    
    then run
    ```
    python3 mac_build.py
    ```
    
- ### On Windows
    **Dependency**

    1. PyInstaller

       Install by `pip3 install pyinstaller`

    then run 

    ```
    &.\win_build.ps1
    ```
